First Git Setup
------

##### Generating SSH Key: 
`$ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"`

##### Cliping SSH Key:
`$ clip < ~/.ssh/id_rsa.pub`

> ##### Adding server origin: 
> `git remote add origin http:\\link-to-repository.com.git`
